Cara Deploy Full Version

Option 1: Deploy ke VPS/Cloud Server

  1. Upload semua files ke server
  2. Jalankan npm install
  3. Jalankan npm start

Option 2: Deploy ke Railway/Render/Heroku

  1. Connect GitHub repository
  2. Set build command: npm install
  3. Set start command: npm start

Option 3: Deploy ke GitHub Pages (Limited)

  1. Upload hanya index.html dan script-github.js
  2. Enable GitHub Pages
  3. Functionality terbatas (no real download)
